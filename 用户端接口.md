## POST 登录接口

POST /user/user/login

用于用户端登录

> Body Parameters

```json
{
  "code": "string"
}
```

### Params

|Name|Location|Type|Required|Title|Description|
|---|---|---|---|---|---|
|body|body|object| no ||none|
|» code|body|string| yes | 微信授权码|需要微信授权码|

> Response Examples

> 成功

```json
{
  "code": 0,
  "msg": "登入成功",
  "data": {
    "id": 21,
    "openid": "ASDKFKNZKJIASDXMZF",
    "token": "ADIXNGSOXKFASDFSASDF.ADFADSFIAODFASDF.DSAFASDIFASDFASDFD"
  }
}
```

> 请求有误

```json
{
  "code": 1,
  "msg": "服务异常请重新登陆",
  "data": null
}
```

### Responses

|HTTP Status Code |Meaning|Description|Data schema|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|成功|Inline|
|400|[Bad Request](https://tools.ietf.org/html/rfc7231#section-6.5.1)|请求有误|Inline|

|---|---|---|---|---|---|
|» code|integer|true|none|响应码|用于判断是否成功0_成功，1_失败|
|» msg|string¦null|false|none|提示信息|后端返回的信息|
|» data|object|true|none||none|
|»» id|integer|true|none||ID 编号|
|»» openid|string|true|none||微信openid|
|»» token|string|true|none||服务端令牌|

|» data|null|false|none||none|

## POST 药品上传接口

POST /user/upload/pharmaceuticals

用户上传图片来获取药品信息

> Body Parameters

```yaml
string

```

### Params

|Name|Location|Type|Required|Title|Description|
|---|---|---|---|---|---|
|character|query|integer| no ||返回完整的字符信息包括字符坐标和候选字等，默认关闭|
|straighten|query|integer| no ||坐标系选项，默认关闭并且所有结果返回均以原图为参照系|
|x-ti-app-id|header|string| yes ||none|
|x-ti-sercret-code|header|string| yes ||none|
|body|body|string(binary)| no ||none|

#### Description

**straighten**: 坐标系选项，默认关闭并且所有结果返回均以原图为参照系
若打开则返回结果的所有坐标点均以正置图像作为参照系
调用者需要注意图像和API结果的角度关系
